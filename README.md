# nodegrpc
### Application based on simple crud operations using node / grpc

## Install packages
> npm i
## Start Server in root directory
> node app.js
## Start any operation in client directory
> node get-news.js <br />
> node selectbyid-news.js <br />
> node add-news.js <br />
> node remove-news.js <br />
> node update-news.js
